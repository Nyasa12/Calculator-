Pink Calculator🌸

A beautiful and beginner-friendly calculator built using Python and CustomTkinter. This project features a soft pink aesthetic, a clean user interface, and supports basic arithmetic operations.

📦 Installation
1. Clone the Repository
git clone https://github.com/your-username/pink-calculator.git
cd pink-calculator
2. Install Dependencies
pip install customtkinter

or

pip install -r requirements.txt
▶️ Run the Application
python calculator.py
🎯 How It Works
Click the buttons to enter numbers and operators.
Press = to calculate the result.
Press C to clear the display.
Invalid expressions will show Error.
